## Step 4: Get Webhook URLs

For EACH channel:
1. Right-click the channel name
2. Click "Edit Channel"
3. Go to "Integrations" tab (left side)
4. Click "Webhooks"
5. Click "New Webhook"
6. Name it (e.g., "garion-orchestration")
7. Click "Copy Webhook URL"
8. Save it somewhere safe

## Webhook URL Format:
```
https://discord.com/api/webhooks/123456789/abcdef123456
```
